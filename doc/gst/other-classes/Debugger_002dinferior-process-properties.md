[]{#Debugger_003a-inferior-process-properties}

#### 4.1.4 Debugger: inferior process properties {#debugger-inferior-process-properties .subsection}

[]{#index-currentLine-1}

**currentLine**

Return the line number in traced process.

[]{#index-isActive-1}

**isActive**

Answer true if the inferior process is still running.

[]{#index-process}

**process**

Answer the inferior process.

[]{#index-suspendedContext}

**suspendedContext**

Answer the suspended execution state of the inferior process.
