### 6.5 Sockets.Datagram {#sockets.datagram .section}

[]{#index-Sockets_002eDatagram}

**Defined in namespace Sockets**\
**Superclass: Object**\
**Category: Sockets-Protocols**

:   This class models a packet of data that is to be sent across the
    network using a connectionless protocol such as UDP. It contains the
    data to be send, as well as the destination address and port. Note
    that datagram packets can arrive in any order and are not guaranteed
    to be delivered at all.

    This class can also be used for receiving data from the network.
