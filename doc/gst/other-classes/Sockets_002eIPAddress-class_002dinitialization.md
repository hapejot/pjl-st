#### 6.12.3 Sockets.IPAddress class: initialization {#sockets.ipaddress-class-initialization .subsection}

[]{#index-createLoopbackHost-1}

**createLoopbackHost**

Answer an object representing the loopback host in the address family
for the receiver. This is 127.0.0.1 for IPv4.

[]{#index-createUnknownAddress-1}

**createUnknownAddress**

Answer an object representing an unkown address in the address family
for the receiver

[]{#index-initialize-8}

**initialize**

Set up the default implementation classes for the receiver
