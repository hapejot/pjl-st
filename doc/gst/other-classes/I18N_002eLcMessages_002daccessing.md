#### 5.10.2 I18N.LcMessages: accessing {#i18n.lcmessages-accessing .subsection}

[]{#index-languageDirectory}

**languageDirectory**

Answer the directory holding MO files for the language

[]{#index-languageDirectory_003a}

**languageDirectory: rootDirectory**

Answer the directory holding MO files for the language, given the root
directory of the locale data.

[]{#index-territoryDirectory}

**territoryDirectory**

Answer the directory holding MO files for the language, specific to the
territory

[]{#index-territoryDirectory_003a}

**territoryDirectory: rootDirectory**

Answer the directory holding MO files for the language, specific to the
territory, given the root directory of the locale data.
