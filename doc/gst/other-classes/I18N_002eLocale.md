### 5.21 I18N.Locale {#i18n.locale .section}

[]{#index-I18N_002eLocale}

**Defined in namespace I18N**\
**Superclass: I18N.LocaleData**\
**Category: i18n-Messages**

:   This object is an abstract superclass of objects related to the
    territory and language in which the program is being used. Instances
    of it are asked about information on the current locale, and provide
    a means to be asked for things with a common idiom, the #? binary
    message.
