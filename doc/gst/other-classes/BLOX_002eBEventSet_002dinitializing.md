#### 1.18.3 BLOX.BEventSet: initializing {#blox.beventset-initializing .subsection}

[]{#index-initialize_003a-1}

**initialize: aBWidget**

Initialize the receiver's event handlers to attach to aBWidget. You can
override this of course, but don't forget to call the superclass
implementation first.
