#### 1.27.1 BLOX.BMenu class: instance creation {#blox.bmenu-class-instance-creation .subsection}

[]{#index-new_003alabel_003a-3}

**new: parent label: label**

Add a new menu to the parent window's menu bar, with 'label' as its
caption (for popup menus, parent is the widget over which the menu pops
up as the right button is pressed).
