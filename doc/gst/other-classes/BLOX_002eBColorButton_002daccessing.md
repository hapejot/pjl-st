**color: aString**

Set the color that the receiver is painted in.

[]{#index-pressed-1}

**pressed**

This is the default callback; it brings up a 'choose-a-color' window
and, if 'Ok' is pressed in the window, sets the receiver to be painted
in the chosen color.
