#### 1.14.3 BLOX.BDropDownList: list box accessing {#blox.bdropdownlist-list-box-accessing .subsection}

[]{#index-index}

**index**

Answer the value of the index option for the widget. Since it is not
possible to modify an item once it has been picked from the list widget,
this is always defined for BDropDownList widgets.
