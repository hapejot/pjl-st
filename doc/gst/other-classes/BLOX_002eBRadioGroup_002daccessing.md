[]{#BLOX_002eBRadioGroup_003a-accessing}

#### 1.38.1 BLOX.BRadioGroup: accessing {#blox.bradiogroup-accessing .subsection}

[]{#index-value-3}

**value**

Answer the index of the button that is currently selected, 1 being the
first button added to the radio button group. 0 means that no button is
selected

[]{#index-value_003a-3}

**value: value**

Force the value-th button added to the radio button group to be the
selected one.
