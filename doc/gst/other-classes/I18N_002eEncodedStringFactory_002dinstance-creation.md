#### 5.4.3 I18N.EncodedStringFactory: instance creation {#i18n.encodedstringfactory-instance-creation .subsection}

[]{#index-encoding-1}

**encoding**

Answer the encoding used for the created Strings.

[]{#index-encoding_003a-3}

**encoding: aString**

Set the encoding used for the created Strings.

[]{#index-fromString_003a-1}

**fromString: aString**

Answer an EncodedString based on aString and in the encoding represented
by the receiver.

[]{#index-new-10}

**new**

Answer a new, empty EncodedString using the encoding represented by the
receiver.

[]{#index-new_003a-9}

**new: size**

Answer a new EncodedString of the given size, using the encoding
represented by the receiver.
