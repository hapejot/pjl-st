#### 1.36.1 BLOX.BProgress: accessing {#blox.bprogress-accessing .subsection}

[]{#index-backgroundColor-9}

**backgroundColor**

Answer the background color of the widget. This is used for the
background of the non-filled part, as well as for the foreground of the
filled part.

[]{#index-backgroundColor_003a-11}

**backgroundColor: aColor**

Set the background color of the widget. This is used for the background
of the non-filled part, as well as for the foreground of the filled
part.

[]{#index-filledColor}

**filledColor**

Answer the background color of the widget's filled part.

[]{#index-filledColor_003a}

**filledColor: aColor**

Set the background color of the widget's filled part.

[]{#index-foregroundColor-8}

**foregroundColor**

Set the foreground color of the widget. This is used for the non-filled
part, while the background color also works as the foreground of the
filled part.

[]{#index-foregroundColor_003a-10}

**foregroundColor: aColor**

Set the foreground color of the widget. This is used for the non-filled
part, while the background color also works as the foreground of the
filled part.

[]{#index-value-1}

**value**

Answer the filled percentage of the receiver (0..1)

[]{#index-value_003a-1}

**value: newValue**

Set the filled percentage of the receiver and update the appearance.
newValue must be between 0 and 1.
