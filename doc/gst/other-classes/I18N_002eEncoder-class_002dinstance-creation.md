#### 5.5.1 I18N.Encoder class: instance creation {#i18n.encoder-class-instance-creation .subsection}

**on: aStringOrStream from: fromEncoding to: toEncoding**

Answer a new encoder that translates from fromEncoding to toEncoding.
The encodings are guaranteed to be those for which the encoder was
registered.
