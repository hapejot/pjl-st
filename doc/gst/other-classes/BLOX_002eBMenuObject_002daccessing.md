[]{#BLOX_002eBMenuObject_003a-accessing}

#### 1.30.1 BLOX.BMenuObject: accessing {#blox.bmenuobject-accessing .subsection}

[]{#index-activeBackground}

**activeBackground**

Answer the value of the activeBackground option for the widget.

Specifies background color to use when drawing active elements. An
element (a widget or portion of a widget) is active if the mouse cursor
is positioned over the element and pressing a mouse button will cause
some action to occur. For some elements on Windows and Macintosh
systems, the active color will only be used while mouse button 1 is
pressed over the element.

[]{#index-activeBackground_003a}

**activeBackground: value**

Set the value of the activeBackground option for the widget.

Specifies background color to use when drawing active elements. An
element (a widget or portion of a widget) is active if the mouse cursor
is positioned over the element and pressing a mouse button will cause
some action to occur. For some elements on Windows and Macintosh
systems, the active color will only be used while mouse button 1 is
pressed over the element.

[]{#index-activeForeground}

**activeForeground**

Answer the value of the activeForeground option for the widget.

Specifies foreground color to use when drawing active elements. See
above for definition of active elements.

[]{#index-activeForeground_003a}

**activeForeground: value**

Set the value of the activeForeground option for the widget.

Specifies foreground color to use when drawing active elements. See
above for definition of active elements.

[]{#index-asPrimitiveWidget-2}

[]{#index-backgroundColor-8}

**backgroundColor**
