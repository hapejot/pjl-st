#### 1.21.1 BLOX.BForm: accessing {#blox.bform-accessing .subsection}

[]{#index-backgroundColor-4}

**backgroundColor**

Answer the value of the backgroundColor option for the widget.

Specifies the normal background color to use when displaying the widget.

[]{#index-backgroundColor_003a-6}

**backgroundColor: value**

Set the value of the backgroundColor option for the widget.

Specifies the normal background color to use when displaying the widget.

[]{#index-defaultHeight}

**defaultHeight**

Answer the value of the defaultHeight option for the widget.

Specifies the desired height for the form in pixels. If this option is
less than or equal to zero then the window will not request any size at
all.

[]{#index-defaultHeight_003a}

**defaultHeight: value**

Set the value of the defaultHeight option for the widget.

Specifies the desired height for the form in pixels. If this option is
less than or equal to zero then the window will not request any size at
all.
