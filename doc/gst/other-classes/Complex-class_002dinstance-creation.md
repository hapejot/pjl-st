#### 2.1.1 Complex class: instance creation {#complex-class-instance-creation .subsection}

[]{#index-i}

**i**

Return the imaginary unit, -1 sqrt.

[]{#index-initialize-1}

**initialize**

Initialize some common complex numbers.

[]{#index-new-7}

This method should not be called for instances of this class.

[]{#index-real_003aimaginary_003a}

**realResult: re imaginary: im**

same generality.
