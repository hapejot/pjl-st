#### 6.20.5 Sockets.SocketAddress class: creating sockets {#sockets.socketaddress-class-creating-sockets .subsection}

[]{#index-newRawSocket}

**newRawSocket**

Create a new raw socket, providing access to low-level network protocols
and interfaces for the protocol family represented by the receiver (for
example, the C protocol family PF_INET for the IPAddress class) Ordinary
user programs usually have no need to use this method.
