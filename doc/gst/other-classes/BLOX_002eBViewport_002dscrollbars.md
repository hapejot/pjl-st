[]{#BLOX_002eBViewport_003a-scrollbars}

#### 1.48.2 BLOX.BViewport: scrollbars {#blox.bviewport-scrollbars .subsection}

[]{#index-horizontal}

**horizontal**

Answer whether an horizontal scrollbar is drawn in the widget if needed.

[]{#index-horizontal_003a}

**horizontal: aBoolean**

Set whether an horizontal scrollbar is drawn in the widget if needed.

[]{#index-horizontalNeeded}

**horizontalNeeded**

Answer whether an horizontal scrollbar is needed to show all the
information in the widget.

[]{#index-horizontalShown}

**horizontalShown**

Answer whether an horizontal scrollbar is drawn in the widget.

[]{#index-vertical}

**vertical**

Answer whether a vertical scrollbar is drawn in the widget if needed.

[]{#index-vertical_003a}

**vertical: aBoolean**

Set whether a vertical scrollbar is drawn in the widget if needed.

[]{#index-verticalNeeded}

**verticalNeeded**

Answer whether a vertical scrollbar is needed to show all the
information in the widget.

[]{#index-verticalShown}

**verticalShown**

Answer whether a vertical scrollbar is drawn in the widget.
