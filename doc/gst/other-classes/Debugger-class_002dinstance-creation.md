#### 4.1.2 Debugger class: instance creation {#debugger-class-instance-creation .subsection}

[]{#index-on_003a-1}

**on: aProcess**

Suspend aProcess and return a new Debugger that controls aProcess.
aProcess must not be the currently running process.
