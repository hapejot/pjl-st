#### 1.36.4 CObject class: subclass creation {#cobject-class-subclass-creation .subsection}

[]{#index-inheritShape-1}

**inheritShape**

Answer whether subclasses will have by default the same shape as this
class. The default is true for the CObject hierarchy.
