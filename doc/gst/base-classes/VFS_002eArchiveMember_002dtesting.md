[]{#VFS_002eArchiveMember_003a-testing}

#### 1.209.8 VFS.ArchiveMember: testing {#vfs.archivemember-testing .subsection}

[]{#index-exists-2}

**exists**

Answer whether a file with the name contained in the receiver does
exist.

[]{#index-isAccessible-3}

**isAccessible**

Answer whether a directory with the name contained in the receiver does
exist and is accessible

[]{#index-isDirectory-3}

**isDirectory**

Answer whether a file with the name contained in the receiver does exist
and identifies a directory.

[]{#index-isExecutable-2}

**isExecutable**

Answer whether a file with the name contained in the receiver does exist
and is executable

[]{#index-isReadable-2}

**isReadable**

Answer whether a file with the name contained in the receiver does exist
and is readable

[]{#index-isSymbolicLink-2}

**isSymbolicLink**

Answer whether a file with the name contained in the receiver does exist
and identifies a symbolic link.

[]{#index-isWriteable-2}

**isWriteable**

Answer whether a file with the name contained in the receiver does exist
and is writeable

[]{#index-mode-2}

**mode**

Answer the octal permissions for the file.

[]{#index-mode_003a-2}

**mode: mode**

Set the octal permissions for the file to be 'mode'.
