#### 1.64.1 Dictionary class: instance creation {#dictionary-class-instance-creation .subsection}

[]{#index-from_003a-5}

**from: anArray**

Answer a new dictionary created from the keys and values of Associations
in anArray, such as {1 -\> 2. 3 -\> 4}. anArray should be specified
using brace-syntax.
