[]{#OrderedCollection_003a-adding}

#### 1.126.3 OrderedCollection: adding {#orderedcollection-adding .subsection}

[]{#index-add_003a-8}

**add: anObject**

Add anObject in the receiver, answer it

[]{#index-add_003aafter_003a}

**add: newObject after: oldObject**

Add newObject in the receiver just after oldObject, answer it. Fail if
oldObject can't be found

[]{#index-add_003aafterIndex_003a}

**add: newObject afterIndex: i**

Add newObject in the receiver just after the i-th, answer it. Fail if i
\< 0 or i \> self size

[]{#index-add_003abefore_003a}

**add: newObject before: oldObject**

Add newObject in the receiver just before oldObject, answer it. Fail if
oldObject can't be found

[]{#index-add_003abeforeIndex_003a}

**add: newObject beforeIndex: i**

Add newObject in the receiver just before the i-th, answer it. Fail if i
\< 1 or i \> self size + 1

[]{#index-addAll_003a-2}

**addAll: aCollection**

Add every item of aCollection to the receiver, answer it

[]{#index-addAll_003aafter_003a}

**addAll: newCollection after: oldObject**

Add every item of newCollection to the receiver just after oldObject,
answer it. Fail if oldObject is not found

[]{#index-addAll_003aafterIndex_003a}

**addAll: newCollection afterIndex: i**

Add every item of newCollection to the receiver just after the i-th,
answer it. Fail if i \< 0 or i \> self size

[]{#index-addAll_003abefore_003a}

**addAll: newCollection before: oldObject**

Add every item of newCollection to the receiver just before oldObject,
answer it. Fail if oldObject is not found

[]{#index-addAll_003abeforeIndex_003a}

**addAll: newCollection beforeIndex: i**

Add every item of newCollection to the receiver just before the i-th,
answer it. Fail if i \< 1 or i \> self size + 1

[]{#index-addAllFirst_003a}

**addAllFirst: aCollection**

Add every item of newCollection to the receiver right at the start of
the receiver. Answer aCollection

[]{#index-addAllLast_003a}

**addAllLast: aCollection**

Add every item of newCollection to the receiver right at the end of the
receiver. Answer aCollection

[]{#index-addFirst_003a-1}

**addFirst: newObject**

Add newObject to the receiver right at the start of the receiver. Answer
newObject

[]{#index-addLast_003a-1}

**addLast: newObject**

Add newObject to the receiver right at the end of the receiver. Answer
newObject
