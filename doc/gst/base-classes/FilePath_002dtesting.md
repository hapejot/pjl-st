[]{#FilePath_003a-testing}

#### 1.77.12 FilePath: testing {#filepath-testing .subsection}

[]{#index-exists-1}

**exists**

Answer whether a file with the name contained in the receiver does
exist.

[]{#index-isAbsolute-3}

**isAbsolute**

Answer whether the receiver identifies an absolute path.

[]{#index-isAccessible-1}

**isAccessible**

Answer whether a directory with the name contained in the receiver does
exist and can be accessed

[]{#index-isDirectory-1}

**isDirectory**

Answer whether a file with the name contained in the receiver does exist
and identifies a directory.

[]{#index-isExecutable-1}

**isExecutable**

Answer whether a file with the name contained in the receiver does exist
and is executable

[]{#index-isFile}

**isFile**

Answer whether a file with the name contained in the receiver does exist
and does not identify a directory.

[]{#index-isFileSystemPath-1}

**isFileSystemPath**

Answer whether the receiver corresponds to a real filesystem path.

[]{#index-isReadable-1}

**isReadable**

Answer whether a file with the name contained in the receiver does exist
and is readable

[]{#index-isRelative}

**isRelative**

Answer whether the receiver identifies a relative path.

[]{#index-isSymbolicLink-1}

**isSymbolicLink**

Answer whether a file with the name contained in the receiver does exist
and identifies a symbolic link.

[]{#index-isWriteable-1}

**isWriteable**

Answer whether a file with the name contained in the receiver does exist
and is writeable
