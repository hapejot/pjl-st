#### 1.72.9 Exception: exception description {#exception-exception-description .subsection}

[]{#index-defaultAction}

**defaultAction**

Execute the default action that is attached to the receiver.

[]{#index-description-2}

**description**

Answer a textual description of the exception.

[]{#index-isResumable-2}

**isResumable**

Answer true. Exceptions are by default resumable.
