#### 1.9.20 Behavior: still unclassified {#behavior-still-unclassified .subsection}

[]{#index-allSharedPoolDictionariesDo_003a}

**allSharedPoolDictionariesDo: aBlock**

Answer the shared pools visible from methods in the metaclass, in the
correct search order.

[]{#index-parseNodeAt_003a}

**parseNodeAt: selector**

Available only when the Parser package is loaded--Answer an RBMethodNode
that compiles to my method named by selector.

[]{#index-updateInstanceVars_003ashape_003a}

**updateInstanceVars: variableArray shape: shape**

Update instance variables and instance spec of the class and all its
subclasses. variableArray lists the new variables, including inherited
ones.
