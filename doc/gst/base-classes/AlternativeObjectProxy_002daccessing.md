#### 1.2.2 AlternativeObjectProxy: accessing {#alternativeobjectproxy-accessing .subsection}

[]{#index-object}

**object**

Reconstruct the object stored in the proxy and answer it. A subclass
will usually override this

[]{#index-object_003a}

**object: theObject**

Set the object to be dumped to theObject. This should not be overridden.

[]{#index-primObject}

**primObject**

Reconstruct the object stored in the proxy and answer it. This method
must not be overridden
