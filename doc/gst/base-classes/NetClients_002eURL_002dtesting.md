[]{#NetClients_002eURL_003a-testing}

#### 1.118.9 NetClients.URL: testing {#netclients.url-testing .subsection}

[]{#index-canCache}

**canCache**

Answer whether the URL is cacheable. The current implementation
considers file URLs not to be cacheable, and everything else to be.

[]{#index-hasFragment}

**hasFragment**

Answer whether the URL points to a particular fragment (anchor) of the
resource.

[]{#index-hasQuery}

**hasQuery**

Answer whether the URL includes query arguments to be submitted when
retrieving the resource.

[]{#index-isFileScheme}

**isFileScheme**

Answer whether the URL is a file URL.

[]{#index-isFragmentOnly}

**isFragmentOnly**

Answer whether the URL only includes the name of a particular fragment
(anchor) of the resource to which it refers.
