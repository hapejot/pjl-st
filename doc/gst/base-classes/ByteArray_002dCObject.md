#### 1.14.4 ByteArray: CObject {#bytearray-cobject .subsection}

[]{#index-asCData}

**asCData**

Allocate memory with malloc for a copy of the receiver, and return a
pointer to it as a CByte.

[]{#index-castTo_003a}

**castTo: type**

Give access to the receiver as a value with the given CType.
