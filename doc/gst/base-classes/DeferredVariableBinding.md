### 1.61 DeferredVariableBinding {#deferredvariablebinding .section}

[]{#index-DeferredVariableBinding}

**Defined in namespace Smalltalk**\
**Superclass: LookupKey**\
**Category: Language-Data types**

:   I represent a binding to a variable that is not tied to a particular
    dictionary until the first access. Then, lookup rules for global
    variables in the scope of a given class are used.
