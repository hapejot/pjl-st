#### 1.69.2 Duration class: instance creation (non ANSI) {#duration-class-instance-creation-non-ansi .subsection}

[]{#index-fromDays_003aseconds_003aoffset_003a-1}

**fromDays: days seconds: secs offset: unused**

Answer a duration of 'd' days and 'secs' seconds. The last parameter is
unused; this message is available for interoperability with the DateTime
class.
