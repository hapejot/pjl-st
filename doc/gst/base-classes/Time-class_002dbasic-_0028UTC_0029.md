[]{#Time-class_003a-basic-_0028UTC_0029}

#### 1.199.1 Time class: basic (UTC) {#time-class-basic-utc .subsection}

[]{#index-midnight}

**midnight**

Answer a time representing midnight in Coordinated Universal Time (UTC)

[]{#index-utcNow}

**utcNow**

Answer a time representing the current time of day in Coordinated
Universal Time (UTC)

[]{#index-utcSecondClock} []{#index-secondClock-1}

**utcSecondClock**

Answer the number of seconds since the midnight of 1/1/1901 (unlike
#secondClock, the reference time is here expressed as UTC, that is as
Coordinated Universal Time).
