[]{#Notification_003a-exception-description}

#### 1.119.1 Notification: exception description {#notification-exception-description .subsection}

[]{#index-defaultAction-1}

**defaultAction**

Do the default action for notifications, which is to resume execution of
the context which signaled the exception.

[]{#index-description-5}

**description**

Answer a textual description of the exception.

[]{#index-isResumable-5}

**isResumable**

Answer true. Notification exceptions are by default resumable.
