### 1.8 Bag {#bag .section}

[]{#index-Bag}

**Defined in namespace Smalltalk**\
**Superclass: Collection**\
**Category: Collections-Unordered**

:   My instances are unordered collections of objects. You can think of
    me as a set with a memory; that is, if the same object is added to
    me twice, then I will report that that element has been stored
    twice.
