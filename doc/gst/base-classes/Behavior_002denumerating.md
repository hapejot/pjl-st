[]{#Behavior_003a-enumerating}

#### 1.9.11 Behavior: enumerating {#behavior-enumerating .subsection}

[]{#index-allInstancesDo_003a}

**allInstancesDo: aBlock**

Invokes aBlock for all instances of the receiver

[]{#index-allSubclassesDo_003a}

**allSubclassesDo: aBlock**

Invokes aBlock for all subclasses, both direct and indirect.

[]{#index-allSubinstancesDo_003a}

**allSubinstancesDo: aBlock**

Invokes aBlock for all instances of each of the receiver's subclasses.

[]{#index-allSuperclassesDo_003a}

**allSuperclassesDo: aBlock**

Invokes aBlock for all superclasses, both direct and indirect.

[]{#index-selectSubclasses_003a}

**selectSubclasses: aBlock**

Return a Set of subclasses of the receiver satisfying aBlock.

[]{#index-selectSuperclasses_003a}

**selectSuperclasses: aBlock**

Return a Set of superclasses of the receiver satisfying aBlock.

[]{#index-subclassesDo_003a}

**subclassesDo: aBlock**

Invokes aBlock for all direct subclasses.

[]{#index-withAllSubclassesDo_003a}

**withAllSubclassesDo: aBlock**

Invokes aBlock for the receiver and all subclasses, both direct and
indirect.

[]{#index-withAllSuperclassesDo_003a}

**withAllSuperclassesDo: aBlock**

Invokes aBlock for the receiver and all superclasses, both direct and
indirect.
