[]{#Character_003a-coercion-methods}

#### 1.28.7 Character: coercion methods {#character-coercion-methods .subsection}

[]{#index-_002a}

**\* aNumber**

Returns a String with aNumber occurrences of the receiver.

[]{#index-asLowercase}

**asLowercase**

Returns self as a lowercase character if it's an uppercase letter,
otherwise returns the character unchanged.

[]{#index-asString-2}

**asString**

Returns the character self as a string. Only valid if the character is
between 0 and 255.

[]{#index-asSymbol}

**asSymbol**

Returns the character self as a symbol.

[]{#index-asUnicodeString-1}

**asUnicodeString**

Returns the character self as a Unicode string.

[]{#index-asUppercase}

**asUppercase**

Returns self as a uppercase character if it's an lowercase letter,
otherwise returns the character unchanged.
