[]{#Exception_003a-accessing}

#### 1.72.5 Exception: accessing {#exception-accessing .subsection}

[]{#index-basicMessageText}

**basicMessageText**

Answer an exception's message text. Do not override this method.

[]{#index-messageText}

**messageText**

Answer an exception's message text.

[]{#index-messageText_003a}

**messageText: aString**

Set an exception's message text.

[]{#index-tag}

**tag**

Answer an exception's tag value. If not specified, it is the same as the
message text.

[]{#index-tag_003a}

**tag: anObject**

Set an exception's tag value. If nil, the tag value will be the same as
the message text.
