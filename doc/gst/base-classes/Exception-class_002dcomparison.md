#### 1.72.1 Exception class: comparison {#exception-class-comparison .subsection}

[]{#index-goodness_003a}

**goodness: anExceptionClass**

Answer how good the receiver is at handling the given exception. A
negative value indicates that the receiver is not able to handle the
exception.

[]{#index-handles_003a}

**handles: anException**

Answer whether the receiver handles 'anException'.
