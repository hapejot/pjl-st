#### 1.69.1 Duration class: instance creation {#duration-class-instance-creation .subsection}

[]{#index-days_003a}

**days: d**

Answer a duration of 'd' days

[]{#index-days_003ahours_003aminutes_003aseconds_003a}

**days: d hours: h minutes: m seconds: s**

Answer a duration of 'd' days and the given number of hours, minutes,
and seconds.

[]{#index-initialize-7}

**initialize**

Initialize the receiver's instance variables

[]{#index-milliseconds_003a}

**milliseconds: msec**

Answer a duration of 'msec' milliseconds

[]{#index-readFrom_003a-2}

**readFrom: aStream**

Parse an instance of the receiver (hours/minutes/seconds) from aStream

[]{#index-weeks_003a}

**weeks: w**

Answer a duration of 'w' weeks

[]{#index-zero}

**zero**

Answer a duration of zero seconds.
