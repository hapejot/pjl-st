#### 1.78.5 FileSegment: printing {#filesegment-printing .subsection}

[]{#index-printedFileName}

**printedFileName**

Answer a printed representation of the file containing the segment.
While introducing some ambiguity, this representation is compact
eliminates the path for kernel files, and produces a relative path from
the current working directory for other files.
