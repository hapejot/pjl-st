[]{#Date_003a-basic}

#### 1.59.4 Date: basic {#date-basic .subsection}

[]{#index-_002d-1}

**- aDate**

Answer a new Duration counting the number of days between the receiver
and aDate.

[]{#index-addDays_003a}

**addDays: dayCount**

Answer a new Date pointing dayCount past the receiver

[]{#index-subtractDate_003a}

**subtractDate: aDate**

Answer the number of days between aDate and the receiver (negative if
the receiver is before aDate)

[]{#index-subtractDays_003a}

**subtractDays: dayCount**

Answer a new Date pointing dayCount before the receiver
