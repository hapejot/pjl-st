#### 1.72.12 Exception: still unclassified {#exception-still-unclassified .subsection}

[]{#index-signalingContext}

**signalingContext**

Return the execution context for the place that signaled the exception,
or nil if it is not available anymore (for example if the exception
handler has returned.
