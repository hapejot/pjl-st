#### 1.155.5 SmallInteger: builtins {#smallinteger-builtins .subsection}

[]{#index-at_003a-20}

**at: anIndex**

Answer the index-th indexed instance variable of the receiver. This
method always fails.

[]{#index-at_003aput_003a-20}

**at: anIndex put: value**

Store value in the index-th indexed instance variable of the receiver
This method always fails.

[]{#index-basicAt_003a-1}

**basicAt: anIndex**

Answer the index-th indexed instance variable of the receiver. This
method always fails.

[]{#index-basicAt_003aput_003a-1}

**basicAt: anIndex put: value**

Store value in the index-th indexed instance variable of the receiver
This method always fails.

[]{#index-scramble}

**scramble**

Answer the receiver with its bits mixed and matched.
