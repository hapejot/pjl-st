[]{#CByte_003a-accessing}

#### 1.20.2 CByte: accessing {#cbyte-accessing .subsection}

[]{#index-cObjStoredType-1}

**cObjStoredType**

Nothing special in the default case - answer the receiver's CType

[]{#index-value-3}

**value**

Answer the value the receiver is pointing to. The returned value is a
SmallInteger

[]{#index-value_003a-3}

**value: aValue**

Set the receiver to point to the value, aValue (a SmallInteger).
