#### 1.29.1 CharacterArray class: basic {#characterarray-class-basic .subsection}

[]{#index-fromString_003a}

**fromString: aCharacterArray**

Make up an instance of the receiver containing the same characters as
aCharacterArray, and answer it.

[]{#index-lineDelimiter}

**lineDelimiter**

Answer a CharacterArray which one can use as a line delimiter. This is
meant to be used on subclasses of CharacterArray.
