#### 1.123.16 Object: storing {#object-storing .subsection}

[]{#index-store}

**store**

Put a String of Smalltalk code compiling to the receiver on the
Transcript (stdout the GUI is not active)

[]{#index-storeLiteralOn_003a-6}

**storeLiteralOn: aStream**

Put a Smalltalk literal compiling to the receiver on aStream

[]{#index-storeNl}

**storeNl**

new line, on the Transcript (stdout the GUI is not active)

[]{#index-storeOn_003a-30}

**storeOn: aStream**

Put Smalltalk code compiling to the receiver on aStream

[]{#index-storeString-1}

**storeString**
