#### 1.40.12 CompiledMethod: source code {#compiledmethod-source-code .subsection}

[]{#index-methodRecompilationSourceString}

**methodRecompilationSourceString**

Answer the method source code as a string, ensuring that it is in new
syntax (it has brackets).

[]{#index-methodSourceCode-2}

**methodSourceCode**

Answer the method source code (a FileSegment or String or nil)

[]{#index-methodSourceFile-2}

**methodSourceFile**

Answer the file where the method source code is stored

[]{#index-methodSourcePos-2}

**methodSourcePos**

Answer the location where the method source code is stored in the
methodSourceFile

[]{#index-methodSourceString-2}

**methodSourceString**

Answer the method source code as a string
