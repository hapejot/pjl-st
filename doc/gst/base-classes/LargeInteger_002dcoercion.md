#### 1.98.5 LargeInteger: coercion {#largeinteger-coercion .subsection}

[]{#index-asCNumber-2}

**asCNumber**

Convert the receiver to a kind of number that is understood by the C
call-out mechanism.

[]{#index-coerce_003a-10}

**coerce: aNumber**

Truncate the number; if needed, convert it to LargeInteger
representation.

[]{#index-generality-4}

**generality**

Answer the receiver's generality

[]{#index-unity-4}

**unity**

Coerce 1 to the receiver's class

[]{#index-zero-5}

**zero**

Coerce 0 to the receiver's class
