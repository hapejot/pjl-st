#### 1.6.6 Association: testing {#association-testing .subsection}

[]{#index-_003d} []{#index-_003d-45}

**= anAssociation**

Answer whether the association's key and value are the same as
anAssociation's, or false if anAssociation is not an Association. As a
special case, identical values are considered equal even if #= returns
false (as is the case for NaN floating-point values).
