[]{#Association_003a-accessing}

#### 1.6.2 Association: accessing {#association-accessing .subsection}

[]{#index-environment}

**environment**

Answer nil. This is present to achieve polymorphism with instances of
VariableBinding.

[]{#index-environment_003a}

**environment: aNamespace**

Do nothing. This is present to achieve polymorphism with instances of
VariableBinding.

[]{#index-key_003avalue_003a-1}

**key: aKey value: aValue**

Set the association's key to aKey, and its value to aValue

[]{#index-value}

**value**

Answer the association's value

[]{#index-value_003a}

**value: aValue**

Set the association's value to aValue
