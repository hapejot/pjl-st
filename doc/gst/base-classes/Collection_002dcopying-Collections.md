#### 1.37.7 Collection: copying Collections {#collection-copying-collections .subsection}

[]{#index-copyReplacing_003awithObject_003a}

**copyReplacing: targetObject withObject: newObject**

Copy replacing each object which is = to targetObject with newObject

[]{#index-copyWith_003a-1}

**copyWith: newElement**

Answer a copy of the receiver to which newElement is added

[]{#index-copyWithout_003a-1}

**copyWithout: oldElement**

Answer a copy of the receiver to which all occurrences of oldElement are
removed
