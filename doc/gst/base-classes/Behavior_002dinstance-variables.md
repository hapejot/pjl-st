[]{#Behavior_003a-instance-variables}

#### 1.9.14 Behavior: instance variables {#behavior-instance-variables .subsection}

[]{#index-addInstVarName_003a}

**addInstVarName: aString**

Add the given instance variable to instance of the receiver

[]{#index-instanceVariableNames_003a}

**instanceVariableNames: instVarNames**

Set the instance variables for the receiver to be those in instVarNames

[]{#index-removeInstVarName_003a}

**removeInstVarName: aString**

Remove the given instance variable from the receiver and recompile all
of the receiver's subclasses
