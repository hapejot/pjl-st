### 1.6 Association {#association .section}

[]{#index-Association}

**Defined in namespace Smalltalk**\
**Superclass: LookupKey**\
**Category: Language-Data types**

:   My instances represent a mapping between two objects. Typically, my
    \"key\" object is a symbol, but I don't require this. My \"value\"
    object has no conventions associated with it; it can be any object
    at all.
