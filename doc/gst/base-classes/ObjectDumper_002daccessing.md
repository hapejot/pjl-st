#### 1.124.5 ObjectDumper: accessing {#objectdumper-accessing .subsection}

[]{#index-flush-1}

**flush**

'Forget' any information on previously stored objects.

[]{#index-stream}

**stream**

Answer the ByteStream to which the ObjectDumper will write and from
which it will read.

[]{#index-stream_003a}

**stream: aByteStream**

Set the ByteStream to which the ObjectDumper will write and from which
it will read.
