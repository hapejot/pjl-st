#### 1.10.2 BindingDictionary: basic & copying {#bindingdictionary-basic-copying .subsection}

[]{#index-_003d-2}

**= arg**

Answer whether the receiver is equal to arg. The equality test is by
default the same as that for equal objects. = must not fail; answer
false if the receiver cannot be compared to arg

[]{#index-hash-2} []{#index-identityHash-1}

**hash**

Answer an hash value for the receiver. This is the same as the object's
#identityHash.
