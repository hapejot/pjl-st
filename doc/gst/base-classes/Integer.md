### 1.92 Integer {#integer .section}

[]{#index-Integer}

**Defined in namespace Smalltalk**\
**Superclass: Number**\
**Category: Language-Data types**

:   I am the abstract integer class of the GNU Smalltalk system. My
    subclasses' instances can represent signed integers of various sizes
    (a subclass is picked according to the size), with varying
    efficiency.
