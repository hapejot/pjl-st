#### 1.75.4 File class: instance creation {#file-class-instance-creation .subsection}

[]{#index-name_003a-2}

**name: aName**

Answer a new file with the given path. The path is turned into an
absolute path.

[]{#index-path_003a}

**path: aString**

Answer a new file with the given path. The path is not validated until
some of the fields of the newly created objects are accessed
