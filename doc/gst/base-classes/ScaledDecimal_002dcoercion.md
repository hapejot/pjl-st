#### 1.148.3 ScaledDecimal: coercion {#scaleddecimal-coercion .subsection}

[]{#index-asCNumber-4}

**asCNumber**

Convert the receiver to a kind of number that is understood by the C
call-out mechanism.

[]{#index-asFloatD-7}

**asFloatD**

Answer the receiver, converted to a FloatD

[]{#index-asFloatE-7}

**asFloatE**

Answer the receiver, converted to a FloatE

[]{#index-asFloatQ-7}

**asFloatQ**

Answer the receiver, converted to a FloatQ

[]{#index-asFraction-4}

**asFraction**

Answer the receiver, converted to a Fraction

[]{#index-ceiling-3}

**ceiling**

Answer the receiver, converted to an Integer and truncated towards
+infinity.

[]{#index-coerce_003a-13}

**coerce: aNumber**

Answer aNumber, converted to a ScaledDecimal with the same scale as the
receiver.

[]{#index-fractionPart-4}

**fractionPart**

Answer the fractional part of the receiver.

[]{#index-generality-6}

**generality**

Return the receiver's generality

[]{#index-integerPart-3}

**integerPart**

Answer the fractional part of the receiver.

[]{#index-truncated-7}

**truncated**

Answer the receiver, converted to an Integer and truncated towards
-infinity.
