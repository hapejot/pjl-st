#### 1.198.4 TextCollector: set up {#textcollector-set-up .subsection}

[]{#index-message-1}

**message**

Answer an association representing the message to be sent to perform
write operations. The key is the receiver, the value is the selector

[]{#index-message_003a-1}

**message: receiverToSelectorAssociation**

Set the message to be sent to perform write operations to the one
represented by anAssociation. anAssociation's key is the receiver, while
its value is the selector
