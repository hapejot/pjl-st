#### 1.118.5 NetClients.URL: copying {#netclients.url-copying .subsection}

[]{#index-copyWithoutAuxiliaryParts}

**copyWithoutAuxiliaryParts**

Answer a copy of the receiver where the fragment and query parts of the
URL have been cleared.

[]{#index-copyWithoutFragment}

**copyWithoutFragment**

Answer a copy of the receiver where the fragment parts of the URL has
been cleared.

[]{#index-postCopy-2}

**postCopy**

All the variables are copied when an URL object is copied.
