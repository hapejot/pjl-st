#### 1.203.6 UnicodeString: multibyte encodings {#unicodestring-multibyte-encodings .subsection}

[]{#index-encoding-3}

**encoding**

Answer the encoding used by the receiver. Conventionally, we answer
'Unicode' to ensure that two UnicodeStrings always have the same
encoding.

[]{#index-numberOfCharacters-1} []{#index-size-28}

**numberOfCharacters**

Answer the number of Unicode characters in the receiver. This is the
same as #size for UnicodeString.
