### 1.40 CompiledMethod {#compiledmethod .section}

[]{#index-CompiledMethod}

**Defined in namespace Smalltalk**\
**Superclass: CompiledCode**\
**Category: Language-Implementation**

:   I represent methods that have been compiled. I can recompile methods
    from their source code, I can invoke Emacs to edit the source code
    for one of my instances, and I know how to access components of my
    instances.
