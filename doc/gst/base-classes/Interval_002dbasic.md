[]{#Interval_003a-basic}

#### 1.93.2 Interval: basic {#interval-basic .subsection}

[]{#index-at_003a-7}

**at: index**

Answer the index-th element of the receiver.

[]{#index-at_003aput_003a-6}

**at: index put: anObject**

This method should not be called for instances of this class.

[]{#index-collect_003a-3}

**collect: aBlock**

Evaluate the receiver for each element in aBlock, collect in an array
the result of the evaluations.

[]{#index-copyFrom_003ato_003a-4}

**copyFrom: startIndex to: stopIndex**

Not commented.

[]{#index-do_003a-4}

**do: aBlock**

Evaluate the receiver for each element in aBlock

[]{#index-isEmpty-3}

**isEmpty**

Answer whether the receiver is empty.

[]{#index-reverse-1}

**reverse**

Answer a copy of the receiver with all of its items reversed

[]{#index-size-10}

**size**

Answer the number of elements in the receiver.

[]{#index-species-1}

**species**

Answer 'Array'.
