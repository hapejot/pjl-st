#### 1.198.1 TextCollector class: accessing {#textcollector-class-accessing .subsection}

[]{#index-message_003a}

**message: receiverToSelectorAssociation**

Answer a new instance of the receiver, that uses the message identified
by anAssociation to perform write operations. anAssociation's key is the
receiver, while its value is the selector.

[]{#index-new-31}

**new**

This method should not be called for instances of this class.
