#### 1.203.2 UnicodeString class: multibyte encodings {#unicodestring-class-multibyte-encodings .subsection}

[]{#index-defaultEncoding}

**defaultEncoding**

Answer the encoding used by the receiver. Conventionally, we answer
'Unicode' to ensure that two UnicodeStrings always have the same
encoding.

[]{#index-isUnicode-6}

**isUnicode**

Answer true; the receiver stores characters.
