[]{#Number_003a-truncation-and-round-off}

#### 1.122.14 Number: truncation and round off {#number-truncation-and-round-off .subsection}

[]{#index-asInteger-2}

**asInteger**

Answer the receiver, rounded to the nearest integer

[]{#index-floor-3}

**floor**

Return the integer nearest the receiver toward negative infinity.

[]{#index-fractionPart-3} []{#index-integerPart-4}

**fractionPart**

Answer a number which, summed to the #integerPart of the receiver, gives
the receiver itself.

[]{#index-integerPart-2}

**integerPart**

Answer the receiver, truncated towards zero

[]{#index-roundTo_003a}

**roundTo: aNumber**

Answer the receiver, truncated to the nearest multiple of aNumber

[]{#index-rounded-2}

**rounded**

Returns the integer nearest the receiver

[]{#index-truncateTo_003a}

**truncateTo: aNumber**

Answer the receiver, truncated towards zero to a multiple of aNumber

[]{#index-truncated-6}

**truncated**

Answer the receiver, truncated towards zero
